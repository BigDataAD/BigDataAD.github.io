BIG DATA Y ANALISIS DE DATOS

¿Qué tanto aprendiste del tema?

  • ¿Qué es Big Data?

    • A.Una gran cantidad de datos que no se pueden procesar utilizando métodos tradicionales.

    B.Una pequeña cantidad de datos que se puede procesar fácilmente.

    C.Un tipo de datos que no es útil para las empresas.

  • ¿Cuál de los siguientes NO es un beneficio del Big Data y el análisis de datos?

    • A.Mejores decisiones basadas en datos.

    B.Mayor eficiencia y productividad.

    C.Peores resultados de los pacientes.

  • ¿Cuál de las siguientes NO es una aplicación del Big Data y el análisis de datos?

    • A.Atención sanitaria

    B.Finanzas

    C.Agricultura

  • ¿Cuál de las siguientes tecnologías NO se utiliza en Big Data y análisis de datos?

    • A.Hadoop

    B.Spark

    C. Microsoft Word.

  • ¿Qué es el “overfitting” en el contexto del aprendizaje automático?

    • A.Es un proceso para reducir el ruido en los datos.

    B.Es un modelo que se ajusta perfectamente a los datos de entrenamiento pero no generaliza bien a datos nuevos.

    • A.Documentales

    B. Relacionales

    C.Clave-valor

    • A.Un método para reducir el tamaño de la muestra.

    B.Un proceso para aumentar la precisión de los datos.

    C.Un enfoque para eliminar valores atípicos.

    B.Regresión logística

    C.PCA (Análisis de Componentes Principales)

    B.Un sistema de gestión de bases de datos relacionales.

    C.Un framework para procesar y almacenar grandes volúmenes de datos distribuidos.

    B.Extracción de características

    C. Evaluación del modelo
